# Pythonでできること

この授業が終わる頃には、皆さんは **自分でプログラムを書き、AIと組み合わせてアプリを作れる** ようになります。

この資料では、これから学ぶPythonで何ができるのかを紹介します。

## 皆さんが使っているAI、その裏側は？

**ChatGPT** に質問したり、**Gemini** に文章を要約させたり、AIで画像や動画を作ったことはありませんか？

実はこれらのサービスは、すべてPythonで作られています。皆さんが毎日触れているAIの裏側で動いているのがPythonです。

Pythonはプログラミング言語の人気ランキングで **世界1位**（[TIOBE Index](https://www.tiobe.com/tiobe-index/)）。AI開発だけでなく、Web開発、セキュリティ、自動化など、IT業界のあらゆる場面で使われています。

## 「AIがコードを書くなら、勉強しなくていい？」

最近では、AIに日本語で指示を出すだけでプログラムを作れる **「バイブコーディング」** が話題になっています。

では、プログラミングを学ぶ意味はもうないのでしょうか？

逆です。**基礎がわかる人ほど、AIを圧倒的にうまく使えます。**

- AIが書いたコードにバグがあったとき、**見抜けるのは基礎がある人だけ**
- 「何を作りたいか」をAIに正確に伝えるには、**プログラミングの知識が必要**
- AIの出力を鵜呑みにせず、**自分で判断・修正できる力** が現場では求められている

AIはツールです。ハサミと同じで、使い方を知っている人が最も力を発揮できます。

## この授業のゴール

1. **前半**: Pythonの基礎文法を身につける（変数、関数、条件分岐、ループなど）
2. **後半**: ネットワーク通信やWebアプリの仕組みを学ぶ（requests、Flask）
3. **最終回**: AIにコードを書かせて、自分で読み解き・修正する「バイブコーディング」を体験

基礎を積み上げた上で、最後にAIと一緒にプログラミングする。**「自分でもできる」** を実感できる授業にしていきます。
