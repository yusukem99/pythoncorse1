`doc_list`に論文の内容のリスト、`keyword`に検索する言葉が渡されます。

検索する言葉`keyword`が含まれる論文のインデックスを含むリストを返すように、以下の`word_search`関数を完成させてください。

ただし、抽出条件は以下のルールに従うものとします。

- 言葉の一部のみが一致したものは、不一致とする。例えばkeyword="closed"のとき、"enclosed."は不一致とする。
- 大文字と小文字を区別しない。例えば、"keyword=closed"のとき"Closed the case."は一致とする。
- ピリオドとカンマは検索条件に影響しない。例えば、keyword="closed"のとき、"It is closed."は一致とする。

例:
```
>>> doc_list = ["The Learn Python Challenge Casino.", "They bought a car", "Casinoville"]
>>> word_search(doc_list, 'casino')
[0]
```
